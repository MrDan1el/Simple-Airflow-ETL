# Simple-Airflow-ETL
Простой ETL процесс с применением ООП подхода.

## Технологический стек и инструменты
- **Airflow**
- **Postgres**
- **Python**
- **SQLAlchemy**

## Структура репозитория
```
Simple-Airflow-ETL/
│
├── config/                             # Конфигурационные файлы проекта
├── dags/                               # DAG'и Airflow
├── model/                              # SQLAlchemy модели
├── scripts/                            # Скрипты проекта
├── utility/                            # Утилиты проекта
│
├── .gitignore                          # Файлы и директории, игнорируемые GIT
└── README.md                           # Обзор проекта
```
