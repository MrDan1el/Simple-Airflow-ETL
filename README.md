# Simple-Airflow-ETL
Простой ETL процесс с применением ООП подхода. 

Данные о погоде для заданных городов извлекаются из API источника и записываются в базу данных Postgres. В проекте реализованы модули для работы с yaml и csv файлами, обработки requests запросов, создания и заполнения таблицы в базе данных.

## Технологический стек и инструменты
- **Airflow**
- **Postgres**
- **Python**
- **SQLAlchemy**

## Структура репозитория
```
Simple-Airflow-ETL/
│
├── dags/                               # DAG'и Airflow
│   ├── from_api_to_pg.py               # DAG для запуска Python скрипта
│
├── src/                                # Исходный код и модули проекта
│   ├── config/                         # Конфигурационные файлы
│   ├── model/                          # SQLAlchemy модели
│   ├── utility/                        # Утилиты проекта
│   ├── run.py                          # Скрипт для запуска
│
├── .gitignore                          # Файлы и директории, игнорируемые GIT
└── README.md                           # Обзор проекта
```
